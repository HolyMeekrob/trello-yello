trello Class

Defined in: lib\trello.js:15
Module: trello-yello

Trello Yello's entry point for accessing the Trello API. It serves as both an object factory for Trello objects as well as a platform for performing operations that aren't associated with specific Trello objects, such as queries against the Trello search engine.

createBoard

(
  • params
)
Object

Defined in lib\trello.js:113

Parameters:

  • params Object

    the method parameters object

    • name String

      the board's name

    • [description] String optional

      the board's description

    • [organizationId] String optional

      the board's organization

    • [cloneId] String optional

      the id of the existing board to clone

    • [keepFromSource='all'] String optional

      a comma-separated list of properties to keep from the source board, or 'all'

    • [powerUps] String optional

      the powerUps to give the new board. This is a comma separated list of values that can include 'calendar', 'cardAging', 'recap', and 'voting'.

    • [permissionLevel='default'] String optional

      the permissions level of the board. Valid values are 'org', 'private', and 'public'.

    • [voting='disabled'] String optional

      the board permissions for voting. Valid values are 'disabled', 'members', 'observers', 'org', and 'public'.

    • [comments='members'] String optional

      the board permissions for commenting. Valid values are 'disabled', 'members', 'observers', 'org', and 'public'.

    • [invitations='members'] String optional

      the board permissions for sending invitations. Valid values are 'admins' and 'members'.

    • [selfJoin=true] Boolean optional

      the board permissions for allowing members to self join

    • [cardCovers=true] Boolean optional

      the board setting for card covers.

    • [background='blue'] String optional

      the board's background

    • [cardAging='regular'] String optional

      the board's card aging setting. Valid values are 'regular' and 'pirate'.

    • [callback] Function optional

      the callback function once the operation is complete. This is optional, as a Promise is returned.

Returns:

Object:

a Promise that resolves to the new board
